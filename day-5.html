Functions

Functions should be familiar to you from math. JavaScript has functions, too but they have a different syntax.

Function from Your Math Class

A function usually involves a variable an a formula. For example, look at this function for a line. 

    f(x) = .25x + 5

At x=0, the function's value is 5. At x=40, the function's value is 15. Pop Quiz: What is the value of the fuction at x=20

The x axis and y axis could represent many things. In this case, we can say x is days and y is cash. If we start with 5 dollars and we sell one ice cream cone a day at $0.25 a piece, our total cash will be $15 after 40 days.

A function is a way to say all of this in one line.

At x=0

    f(0) = 0 * .25 + 5
    f(0) = 5

At x=40

    f(40) = 40 * .25 + 5
    f(40) = 10 + 5
    f(40) = 15

POP QUIZ: what is the value at x=20?

    f(20) = 20 * .25 + 5
    f(20) = ?

Our ice cream shop raised it's prices to $5 for one cone. We still sell one cone a day. We still start with $5. Our new function looks like this: 

    g(x) = 2 * x + 5;

Now at 5 days, g(x) will be 

    g(x) = 5 * 5 + 5

    g(x) = 30

    g(x) = 50 + 5
